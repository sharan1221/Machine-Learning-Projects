# Machine Learning Projects

A collection of machine learning projects covering supervised learning,
unsupervised learning, and computer vision.

## Projects

### 🏠 House Price Prediction
Linear Regression model trained on 1,460 real houses to predict sale prices.
**R² Score: 0.81 | RMSE: $37,865**
→ [View Project](./House-Price-Prediction)

---

### 🛍️ Customer Segmentation
K-Means Clustering to group 200 mall customers into 5 meaningful segments.
**5 clusters | Elbow Method used to find optimal K**
→ [View Project](./Customer-Segmentation)

---

### 🐱🐶 Cat vs Dog Image Classifier
SVM trained on 2,000 images to classify cats and dogs using pixel data.
**~72% accuracy | RBF kernel | OpenCV preprocessing**
→ [View Project](./Cat-Dog-Image-Classifier)

---

### 🖐️ Hand Gesture Recognition
SVM + HOG features to classify 10 different hand gestures from images.
**10 gesture classes | HOG feature extraction | One-vs-Rest SVM**
→ [View Project](./Hand-Gesture-Recognition)
